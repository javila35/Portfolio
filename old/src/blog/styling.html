Styling My Portfolio

First Post: Jan 9, 2021

I am in the process of redoing my portfolio. I scrapped my original portfolio that I built with Gatsby in favor or building a site from vanilla HTML, CSS, and JS. I miss when the internet was a bit simpler. Eventually I'd like to end up with a very interactive creative site. For example, some of my favorites are Good.Brand, Toy Fight, and Tommy Genesis' website.

Those websites use some very cool animation and integrate the user's action into the feel of the website. Eventually I'd like to be working on projects that incorporate that. I'm especially keen on integrating user interaction into the website to create a novel experience. I'd also like to incorporate my personal interests into it somehow. The books I read, the shows and movies I watch, the music I listen to it. Just throwing ideas out there for myself.

As it stands, I am just creating the barebones structure of the website. I'm still not 100% sure what all I want to include. But in the interim between having a fully formed website with elements of interaction and creativity, I still want this website to be easy to navigate and pleasing to look at.

I started out with the typing. I was listening ot a SyntaxFM episode where they talk about type systems. I had never really thought about a typography CSS file. Because my site doesn't have much content, implementing a consistent font style across the site now is easier than it will be down the line.
